User Guide

How GridGen Works

GridGen lays a labeled coordinate grid over any image and exports the result as a print-ready PDF — entirely in your browser, with no upload, account, or install.

Overview

GridGen is a single-page web app. You give it an image, it draws a grid on top of a live preview, and you tune the grid until it reads well against your picture. When you are happy, it renders everything to a PDF and downloads it to your device.

Your images never leave your device. GridGen reads the file locally in the browser and does all drawing on an HTML5 canvas. Nothing is uploaded to a server — see the Technical Reference for details.

The workflow

  1. Upload an image (PNG, JPG, or WEBP).
  2. Pick a grid mode and set columns/rows or a cell size.
  3. Adjust the appearance — color, thickness, opacity — until the grid is legible.
  4. Configure labels — style, position, and origin.
  5. Download the PDF.

Uploading an image

There are two ways to load an image:

  • Click the upload zone in the sidebar and pick a file from your device.
  • Drag and drop a file onto the preview area — the large panel on the right is the drop target.

Supported formats are PNG, JPG, and WEBP. The original pixel dimensions are preserved throughout, so the exported grid lines up exactly with your source image. Once loaded, the file name is shown and you can hit Replace to swap in a different image at any time.

Grid modes

GridGen offers two ways to define the grid. Switch between them with the By cells / By size tabs.

ModeYou setBest for
By cellsNumber of columns and rows (1–100 each)A fixed grid shape regardless of image size — e.g. a 10 × 10 drawing grid.
By sizeA single cell size in pixels (5–500)Consistent real-world scale — cells stay the same pixel size and GridGen fits as many as needed.

In By cells mode, cell width and height are the image dimensions divided by the column and row counts, so cells may be rectangular. In By size mode, cells are square and the last row/column is clipped to the image edge if it does not divide evenly.

Appearance

Three controls govern how the grid lines look:

  • Color — choose from presets (black, white, red, blue, green) or open the custom color picker. For dark images, switch the line color to white.
  • Line thickness — 1 to 10 pixels.
  • Opacity — 10% to 100%, so the grid can sit lightly over detailed images.

The live preview updates instantly as you drag the sliders, so you can dial in readability before exporting.

Coordinate labels

Toggle the Coordinates switch to add cell labels. Three options shape how they appear:

Style

  • Chess (A1) — columns are letters (A, B, C… and AA, AB after Z), rows are numbers.
  • Numeric (1,1) — both axes are numbers, written as column,row.

Position

  • Outside — column labels sit in a clean white margin above the image and row labels in a margin to its left. Most readable for printing and reference.
  • Inside — labels are placed in the corner of each cell with an automatic contrast halo so they stay legible over any image content.

Origin

  • Top-left — row 1 is at the top. Matches image-pixel and spreadsheet conventions.
  • Bottom-left — row 1 is at the bottom. Matches Cartesian coordinates and chess boards.
Tip: the label color adjusts automatically for legibility. When you pick a light line color, GridGen darkens the outside labels and flips the inside-label halo so text never washes out.

Exporting a PDF

Click Download PDF. GridGen renders the full-resolution canvas — image, grid, and labels — into an A4 page:

  • Orientation is chosen automatically (landscape or portrait) to match your image's aspect ratio.
  • The artwork is scaled to fit the page with a small margin and centered.
  • The file is named after your source image — for example, site-photo becomes site-photo-grid.pdf.

Because the grid is drawn at the image's native resolution, fine lines stay crisp when printed.

Use cases

  • Artists — use the classic grid drawing method to transfer or scale up a reference image by hand.
  • Game masters — turn a map or illustration into a battle grid with coordinate references.
  • Orienteering & scouting — add a coordinate reference grid to a map image and print it. See grids on maps.
  • Archaeologists & field researchers — overlay a reference grid on site photos for documentation.
  • Teachers & students — create coordinate-plane worksheets and labeled diagrams.

For the full set of use cases with the recommended settings for each, see ways to use GridGen.

FAQ

Is GridGen free?

Yes. It is free, open source, and released under the GNU GPL v3.

Do my images get uploaded anywhere?

No. All processing happens locally in your browser. Your images are never sent to a server.

What image formats are supported?

PNG, JPG, and WEBP — any raster format your browser can decode.

What resolution is the exported PDF?

The grid is composited at your image's full native resolution, then placed on an A4 page scaled to fit.

Which browsers work?

Any modern browser with HTML5 Canvas and File API support — Chrome, Firefox, Safari, and Edge.
